## 📌 Project Description

This is a Machine Learning-based application that translates French words into Tamil.

The system only translates French words that contain exactly **5 letters**.  
If the entered word has fewer or more than 5 letters, the system will not translate it.

The application includes a graphical user interface (GUI) built using Tkinter.

---

## 🎯 Objective

- To implement a simple Machine Translation system
- To apply a condition-based filter (5-letter validation)
- To design a user-friendly GUI for input and output
- To demonstrate the use of NLP and pre-trained models

---

## 🛠 Technologies Used

- Python
- Tkinter (GUI)
- Transformers Library
- Torch
- SentencePiece

---

## ⚙️ Features

- GUI-based interface
- Accepts French word as input
- Validates word length (exactly 5 letters)
- Translates into Tamil
- Displays error message for invalid input
- Simple and easy to use

---

## 🧠 How It Works

1. User enters a French word.
2. Program checks if the word has exactly 5 letters.
3. If condition is satisfied:
   - The ML model processes the input.
   - Tamil translation is displayed.
4. If not:
   - Error message is shown.
